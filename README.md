# Diddy Mobile

A mobile application built using React Native,

## Requirements

- NodeJS
- Xcode

## Setup instructions

### 1. Install dependencies
```sh
# Clone the repo
git clone https://github.com/DiddyApp/diddy-mobile

cd diddy-mobile

# Install npm dependencies
yarn

# Setup App
yarn app:setup
```
## Usage
```ssh
# start application
yarn ios
```
